## Program Analisis Nilai Mahasiswa

Program Python sederhana yang mengimplementasikan konsep:

### 1. Struktur Kontrol
- **if-elif-else**: Menentukan grade dan status kelulusan mahasiswa
- **for loop**: Iterasi data mahasiswa dan menampilkan hasil
- **while loop**: Menampilkan soal ujian yang diacak

### 2. Struktur Data
- **Dictionary**: Menyimpan data nama dan nilai mahasiswa
- **List**: Menyimpan daftar nilai dan hasil analisis
- **Tuple**: Menyimpan batas nilai grade

### 3. Library
- **statistics**: Menghitung rata-rata (mean), median, dan standar deviasi nilai
- **random**: Mengacak soal ujian dari bank soal
- **datetime**: Menampilkan waktu analisis program

## Cara Menjalankan

```bash
python analisis_nilai_mahasiswa.py
```

## Output Program

Program akan menampilkan:
1. Tabel analisis nilai setiap mahasiswa (rata-rata, median, stdev, grade, status)
2. Ringkasan kelas (rata-rata kelas, nilai tertinggi/terendah, persentase kelulusan)
3. Simulasi pengacakan soal ujian Kecerdasan Buatan
